# Daily 6

**6 habits. Every day.** A minimal habit tracker. Data is stored in your browser (no account required).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build & deploy

```bash
npm run build
```

See **SETUP.md** for moving this project to its own folder and setting up a fresh Git repo, then deploying (e.g. Vercel).
